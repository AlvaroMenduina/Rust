# Installation

Here is the ``rustup`` notes (https://rust-lang.github.io/rustup/)

You can check the current version of rust by running this in the terminal
```
rustc --version
```

## Running code

Code is compiled with
```
rustc hello.rs
```

and runs with

```rust
./hello
```
